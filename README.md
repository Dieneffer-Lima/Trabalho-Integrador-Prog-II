Para executar o projeto é necessário instalar o Node.js e o PostgreSQL. No backend devem ser instaladas as dependências express, cors, dotenv, sequelize, pg, pg-hstore, passport, passport-jwt, jsonwebtoken, bcrypt, nodemon e axios quando necessário, todas instaladas automaticamente ao executar npm install dentro da pasta backend. No frontend devem ser instaladas as dependências react, react-dom, axios e as bibliotecas padrão do Vite ao executar npm install dentro da pasta frontend. Após instalar, é preciso criar no PostgreSQL o banco borracharia_db e configurar no backend um arquivo .env contendo DB_NAME=borracharia_db, DB_USER=postgres, DB_PASS=postgres, DB_HOST=localhost, JWT_SECRET=chave e PORT=3001. O backend é iniciado com npm start e o frontend com npm run dev, e o sistema estará funcionando normalmente após ambos os serviços estarem ativos.
